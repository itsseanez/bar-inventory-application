# bar-inventory-application
A full-stack inventory application designed for managing bar stock, tracking quantities of drinks, and organizing inventory by category. Built with Node.js, Express, and PostgreSQL, this project demonstrates relational database design, RESTful routing, and server-rendered views.
## Features
- Track bar inventory items (liquor, mixers, etc.)
- Organize items by category
- View all inventory items and categories
- Add, update, and delete inventory items
- Maintain and update quantities
- Persistent data using PostgreSQL
- Server-rendered UI using EJS
## Tech Stack
- Backend: Node.js, Express
- Database: PostgreSQL
- View Engine: EJS
